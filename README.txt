This repository has the files and database for Srijan's project TwitterQueue.In the websites current state the User's authentication is done using twitter login. Then the user is able to schedule tweets according to a specified time and date or periodically with a fixed interval between the tweets. The user's tweets are also visible under 'YOUR TWEETS'. 
Further features to be added to the website include:
1) the ability of user to create a list and shedule the list instead of individual tweets
2) creating the list of older tweets and reposting them on twitter repeatedly by adding a scheduler to the list
   [wordpress plugin revive tweets/post old tweets can be seen for reference]
   [vbo is helpful for selection of tweets from old list but a custom action is to be made]
3) creating multiple tweet list
4) adding multiple times to a list
